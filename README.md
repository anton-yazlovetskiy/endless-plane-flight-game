# ✈️ Sky Runner

Бесконечный 3D раннер — управляй бумажным самолётиком, уклоняйся от препятствий!

## 🎮 Играть

**Онлайн:** Просто откройте `index.html` в браузере

**На телефоне:** Откройте файл в мобильном браузере, управление касанием левой/правой половины экрана

## 🕹️ Управление

| Действие | Клавиатура | Телефон |
|----------|------------|---------|
| Влево | ← или A | Тап левой половины экрана |
| Вправо | → или D | Тап правой половины экрана |
| Ускорение | Пробел (удерж.) | — |
| Меню | M | Кнопка ⚙ |
| Пауза | P | Через меню |

## 📱 Установка на Android

### Способ 1: Как веб-приложение (рекомендуется)

1. Откройте `index.html` в Chrome на телефоне
2. Нажмите **⋮** → **Добавить на главный экран**
3. Готово! Иконка появится как у обычного приложения

### Способ 2: Собрать APK

1. Установите [Android Studio](https://developer.android.com/studio)
2. Скопируйте игру в assets:
   ```bash
   mkdir -p android/app/src/main/assets
   cp index.html android/app/src/main/assets/
   ```
3. Откройте папку `android/` в Android Studio
4. **Build → Build APK(s)**
5. APK будет в `android/app/build/outputs/apk/debug/`

## 🌐 Публикация на GitHub Pages

### Шаг 1: Создайте репозиторий

```bash
git init
git add index.html README.md
git commit -m "Sky Runner game"
```

### Шаг 2: Загрузите на GitHub

1. Создайте новый репозиторий на [github.com/new](https://github.com/new)
2. Назовите его, например, `sky-runner`
3. Выполните:

```bash
git remote add origin https://github.com/ВАШЕ_ИМЯ/sky-runner.git
git branch -M main
git push -u origin main
```

### Шаг 3: Включите GitHub Pages

1. Зайдите в репозиторий → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / **root**
4. Нажмите **Save**

### Шаг 4: Готово!

Через 1-2 минуты игра будет доступна по адресу:
```
https://ВАШЕ_ИМЯ.github.io/sky-runner/
```
